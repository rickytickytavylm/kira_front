# Кира — веб (фронтенд)

Статичный сайт-чат Киры (HTML/CSS/JS, без сборки). Дизайн в стиле Framer/Apple, окно общения — как у Gemini. Кладётся на GitHub Pages.

## Файлы
- `index.html` — разметка (hero + чат)
- `styles.css` — стили (Apple/Framer, тёмная/светлая тема, адаптив)
- `app.js` — логика чата и стриминг ответа (SSE)
- `config.js` — **адрес бэкенда** (единственное, что нужно настроить)

## Настройка
Открой `config.js` и укажи адрес задеплоенного бэкенда:
```js
BACKEND_URL: "https://твой-бэкенд.up.railway.app"
```
Локально (когда открываешь через localhost) автоматически используется `http://localhost:8787`.

## Локальный запуск
Любой статичный сервер, например:
```bash
python -m http.server 5500
# затем открой http://localhost:5500
```
(Просто открыть файл через file:// не даст ходить в бэкенд из-за CORS — нужен http.)

## Деплой на GitHub Pages
1. Запушь содержимое этой папки в отдельный репозиторий (или ветку).
2. Settings → Pages → Source: ветка с этими файлами, папка `/root`.
3. Файл `.nojekyll` уже добавлен, чтобы Pages не обрабатывал сайт через Jekyll.
4. В `config.js` пропиши боевой адрес бэкенда, а на бэкенде в `ALLOWED_ORIGINS` добавь домен Pages.
